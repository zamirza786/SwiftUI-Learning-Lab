# SwiftUI Fundamentals — Swift 6 Edition 🚀

A collection of SwiftUI projects built while revisiting and strengthening my SwiftUI fundamentals through Sean Allen's SwiftUI course, updated for modern Swift and SwiftUI development.

## About

Even after working hands-on with SwiftUI for about a year, I felt the need to revisit the fundamentals and gain a deeper understanding of how SwiftUI behaves under the hood.

Over 7 days, I rebuilt and modernized several projects while refreshing core concepts such as:

* SwiftUI Lifecycle
* @State
* @StateObject
* @FocusState
* onChange
* Custom Modifiers
* State Management
* Navigation
* API Integration
* MVVM Principles
* SwiftUI Previews
* Modern SwiftUI Best Practices

This repository serves as both a learning resource and a modern reference implementation for developers looking to strengthen their SwiftUI foundations.

## Included Projects

### 🍽️ Appetizers

A food ordering application demonstrating:

* Networking
* Async data loading
* Grid layouts
* Image loading
* State management
* Reusable SwiftUI components

### 🍎 Apple Frameworks

Explore popular Apple frameworks through a clean SwiftUI interface.

Concepts covered:

* Navigation
* Sheet presentation
* Grid layouts
* Data modeling
* Reusable views

### 📋 Apple Frameworks List

A list-based variation of the Apple Frameworks project showcasing:

* Lists
* NavigationStack
* View composition
* Data-driven UI

### 📷 Barcode Scanner

A barcode scanning application built using AVFoundation.

Concepts covered:

* Camera access
* AVFoundation integration
* SwiftUI/UIKit interoperability
* Real-time scanning
* State-driven UI updates

### 🌤️ SwiftUI Weather

A weather application demonstrating:

* Weather data presentation
* Dynamic UI updates
* Custom styling
* View composition
* State management

### 🎨 iOS Frameworks Icons

A framework browser featuring framework icons and visual exploration of Apple's ecosystem.

Concepts covered:

* Grid layouts
* Assets management
* Navigation
* SwiftUI design patterns

## Why This Repository Exists

Some resources used in the original course have changed over time.

To make the learning experience smoother for developers today, I:

* Updated projects for modern Swift and SwiftUI
* Replaced outdated or expired resources
* Recreated missing image assets where necessary
* Fixed compatibility issues
* Ensured projects build successfully on current Xcode versions
* Kept all projects functional and easy to explore

Instead of watching an entire course and recreating everything line-by-line, developers can use these projects as working reference implementations while learning the underlying concepts.
